Confira e utilize o projeto no <a href="https://danielle-luz.github.io/Css-variables-generator/">github pages</a>.


<h2 id="descricao">:pencil2: Descrição</h2>
Site onde é possível informar uma paleta de cores e as medidas de fontes utilizadas no projeto, após interagir com um botão, é possível obter variáveis CSS com os dados.

<h2 id="funcionalidades">:hammer: Funcionalidades</h2>

- Seleção de paleta com base em grupos de cores: brand, gray scale e status.
- Conversão das fontes informadas em px para rem.
- Cópia das variáveis CSS nomeadas para a área de transferência.
- Salvamento de estilos no localStorage para que sejam exibidos quando a página for acessada novamente.
